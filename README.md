## Features

- Radix UI Primitives
- Tailwind CSS
- Fonts with `next/font`
- Icons from [Lucide](https://lucide.dev)
- Dark mode with `next-themes`
- Automatic import sorting with `@ianvs/prettier-plugin-sort-imports`
- Tailwind CSS class sorting, merging and linting.
